This is a custom build of Fluid Infusion 1.3 including: json, fss, framework, renderer, inlineEdit, pager and jQueryUIWidgets.

It was built from commit e57ecdbe05d0b6d097c0b0d5c3b4d57ee6f0d680 of Infusion project master, at
    http://github.com/fluid-project/infusion

using the following command:
   ant customBuild -Dinclude="json, fss, framework, renderer, inlineEdit, pager, jQueryUIWidgets" -Djsfilename="CSpaceInfusion.js" -DnoMinify="true"
